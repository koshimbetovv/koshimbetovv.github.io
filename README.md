# Doniyor Koshimbetov — Personal Academic Webpage

This is a static personal academic webpage designed for GitHub Pages.

## Contents

- `index.html` — main webpage
- `assets/css/styles.css` — styling
- `assets/js/main.js` — mobile navigation, active section highlighting, and dark-mode toggle
- `assets/img/profile.jpeg` — profile photo
- `assets/docs/Academic_CV.pdf` — downloadable CV

## How to publish on GitHub Pages

1. Create a new GitHub repository named:

   ```text
   koshimbetovv.github.io
   ```

2. Upload all files from this folder into that repository.

3. Go to:

   ```text
   Settings → Pages
   ```

4. Under **Build and deployment**, choose:

   ```text
   Source: Deploy from a branch
   Branch: main
   Folder: /root
   ```

5. Save. Your page should be available at:

   ```text
   https://koshimbetovv.github.io
   ```

## How to edit

- Edit text directly in `index.html`.
- Edit colors, spacing, layout, and responsiveness in `assets/css/styles.css`.
- Replace the profile image by overwriting `assets/img/profile.jpeg`.
- Replace the CV by overwriting `assets/docs/Academic_CV.pdf`.
